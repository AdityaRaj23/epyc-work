## Tech Stack

**Client:** Next, React, TailwindCSS


## Run Locally

Clone the project

```bash
  git clone https://github.com/AdityaRaj23/epyc-work.git
```

Go to the project directory

```bash
  cd epyc-work
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
